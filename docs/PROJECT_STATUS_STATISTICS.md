## Known Limitations

### v1.5.3 Limitations

1. **Layer 8 (Deep)** requires high memory (2GB+)
   - **Workaround**: Use FAST/MEDIUM paths for low-memory systems

2. **FINANCIAL_ARCHIVE** pattern collapse is domain-specific
   - **Workaround**: Use DATACENTER_GENERAL for generic data

3. **EXPERIMENTAL_RND** not guaranteed stable
   - **Workaround**: Use only for research, not production
